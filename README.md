# Инструкции

1. Версия Node.js: v14.15.0
### Проверка
```bash
nvm list
nvm install v14.15.0
nvm use v14.15.0
```
2. Установка зависимостей:
```bash
npm install
```
3. Выполните сборку проекта:
```bash
npm run build
```
4. Запустите проект:
```bash
npm run start
```
